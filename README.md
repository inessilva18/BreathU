#  Assistente Inteligente - IACD UC

Sistema multiagente de gestão de tempo e bem-estar desenvolvido para a disciplina de **Inteligência Artificial e Resolução de Problemas** (LIACD, Universidade de Coimbra).

##  Funcionalidades
- Chat por texto e voz
- Gestão de tarefas e calendário pessoal
- Registo de emoções e bem-estar
- Relatórios automáticos de produtividade e humor

##  Executar localmente
```bash
pip install -r requirements.txt
streamlit run app.py
